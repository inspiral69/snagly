# Snagly

AI resale assistant — finds undervalued marketplace listings, estimates real profit after fees and shipping, and alerts you when a strong opportunity appears.

See [CONCEPT.md](./CONCEPT.md) for the vision, [DESIGN.md](./DESIGN.md) for brand/UI, and [ROADMAP.md](./ROADMAP.md) for the feature list and build plan.

## Status

**Phase 0 mostly done** — see [AGENTS.md](./AGENTS.md) for the live handoff (stack, decisions, what’s next).

Launch niche: **telescopes** on **eBay UK**.

## Run the app

```bash
cd mobile
npm install
npm run web          # quickest to preview
# or: npm start      # then open iOS Simulator / Expo Go
```
